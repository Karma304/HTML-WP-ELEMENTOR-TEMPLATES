## üöÄ Quick Start Guide

### For WordPress Elementor Users:

1. **Choose Your Widget**
   - Browse the catalog above
   - Select widgets that match your needs

2. **Copy the Code**
   - Open the widget file from the appropriate folder
   - Copy all content (HTML + CSS + JS if present)

3. **Add to Elementor**
   - Open Elementor editor
   - Add an HTML widget
   - Paste the copied code
   - Customize text, images, and colors

4. **Customize**
   - Update placeholder text
   - Replace image sources
   - Modify colors in the `<style>` section
   - Adjust spacing as needed

### For Developers:

1. **Clone or Download**
   ```bash
   git clone https://github.com/Karma304/HTML-WP-ELEMENTOR-TEMPLATES.git
   ```

2. **Browse Categories**
   - Navigate to `/widgets/` folder
   - Explore category subfolders
   - Review widget code

3. **Integrate**
   - Copy widget HTML
   - Include in your project
   - Customize styling
   - Add functionality

## üé® Customization Guide

### Changing Colors

Find and replace in the `<style>` section:
- **Primary Blue**: `#0066ff` ‚Üí Your brand color
- **Success Green**: `#10b981` ‚Üí Your success color
- **Error Red**: `#ef4444` ‚Üí Your error color
- **Background**: `#f8f9fa` ‚Üí Your background color

### Adjusting Spacing

Common spacing values:
- Section padding: `padding: 5rem 0;` ‚Üí Adjust vertical space
- Container max-width: `max-width: 1200px;` ‚Üí Adjust content width
- Grid gap: `gap: 2rem;` ‚Üí Adjust spacing between items

### Updating Content

- Replace `src="placeholder.jpg"` with your image paths
- Update text content between HTML tags
- Modify links in `href` attributes
- Change form action URLs
